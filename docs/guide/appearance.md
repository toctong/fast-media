# 外观与主题

入口：**设定 → 系统设置 → 个人设置**（及看板娘相关 Tab）

主题与粒子为**本机即时生效**（浏览器本地保存），换设备需重新选择。

## 个人资料

| 项 | 说明 |
|----|------|
| 头像 / 昵称 | 展示用 |
| 用户名 / 改密 | 登录账号安全 |

## 外观主题

### 粒子动态

可选风格示例（名称以界面为准）：星尘漫步、深空星野、极光流彩等。点击即切换。

### 风格背景

可选风格示例：流体溢彩、层叠海浪、律动点阵等。与粒子可组合。

建议：

- 追求干净：粒子弱一点的「星尘 / 深空」+ 低动态背景  
- 追求氛围：极光 / 流体类

## 看板娘（可选）

| 项 | 说明 |
|----|------|
| 启用看板娘 | 总开关 |
| 听歌律动 | 随音乐摆动 |

## 文档站与产品站

本说明文档使用独立深色主题；**产品内主题**请在上述设定页调整，两者互不影响。
