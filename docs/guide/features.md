# 功能总览

一眼对照：模块 → 能干什么 → 去哪配置。

| 模块 | 能力 | 配置文档 |
|------|------|----------|
| 概览 | 媒体统计、继续观看、天气 / 一言 | [初始化](./setup.md) |
| 探索 | 热榜、详情、搜片 | [初始化](./setup.md)（TMDB） |
| 订阅 / RSS | 追更、站点、Transmission、规则、日历 | [RSS](./rss.md) |
| 整理 & 刮削 | 重命名、STRM、归档、分类 | [整理与 STRM](./organize.md) |
| 文件管理 | 本地 / 网盘目录浏览 | [CD2](./cd2.md) · [115](./plugins.md) |
| 插件中心 | 115 / CD2 / Emby / 频道 / 概览插件 | [插件](./plugins.md) |
| Emby 302 | 直链代理、路径映射、扫库 | [Emby](./emby.md) |
| 频道资源 | QQ / TG 频道 | [频道与通知](./channels.md) |
| 通知 | 企业微信 / ClawBot | [频道与通知](./channels.md) |
| 智能助手 | LLM + 系统工具 | [插件](./plugins.md) |
| 外观 | 粒子 / 背景 / 看板娘 | [外观与主题](./appearance.md) |
