# 插件配置

入口：**插件中心**

按下方表格勾选你需要的能力；本文不含第三方资源站插件。

## 插件一览

| 插件 | 做什么 | 关键配置 |
|------|--------|----------|
| **115助手 · 账号** | 登录 115、空间 / VIP | 扫码或 Cookie：`UID; CID; SEID` |
| **115 · 云下载** | 磁力推离线 | 账号、默认保存目录 |
| **115 · 转存** | 分享链接转存 | 账号、默认目录 |
| **115 · 回收站** | 定时清空 | 账号、回收站密码、Cron（默认每天 04:00） |
| **CloudDrive2** | 网盘挂载对接 | 见 [CD2](./cd2.md) |
| **Emby** | 媒体库 / 302 / 扫库 / 深度删除 | 见 [Emby](./emby.md) |
| **频道** | QQ / TG 频道资源 | 见 [频道与通知](./channels.md) |
| **天气** | 概览天气 | 城市、定时推送 |
| **每日一言** | 概览金句 | 分类、定时推送 |
| **智能助手** | LLM + 系统工具 | 设定 → 智能助手（见下） |

## 115助手

### 账号

1. 打开 **115助手 → 账号**  
2. 扫码登录，或手动粘贴 Cookie  
3. 填写名称并保存，确认空间显示正常

### 云下载

| 字段 | 说明 |
|------|------|
| 启用 | 开 |
| 账号 | 选已登录账号 |
| 默认保存目录 | 离线任务落盘位置 |

探索 / 订阅推磁力到 115 时依赖此项。

### 转存 / 回收站

- **转存**：默认目录 + 粘贴分享链接 / 访问码预览  
- **回收站**：密码 + Cron，定期清空  

## 智能助手（设定 → 智能助手）

| 项 | 说明 |
|----|------|
| 启用 / 全局悬浮 / 思考模式 | 交互开关 |
| 系统工具 (MCP) | 读库、搜资源、推下载、跑任务 |
| 允许执行操作 | 写操作需谨慎开启 |
| LLM 提供商 / 基础 URL / API Key / 模型 | OpenAI 兼容即可 |
| 助手名称、提示词、上下文长度 | 按需 |

## 概览小插件

- **天气**：城市（默认北京）+ 是否定时推送  
- **每日一言**：分类 + 定时推送  

在概览页确认展示正常即可。
